# 更新记录

## 2026-05-09

- 调整界面配色为接近软件图标的克制铜橙色，收敛圆角和边框，减少高饱和橙色的视觉干扰。
- 提亮强调色，使用更接近软件图标的专业橙色，并固定水印来源区域高度，避免切换“文字”时下方控件位移。
- 将界面基调调整为白色和浅暖灰，橙色只用于强调、选中态和重点文字；将水印来源区改为固定高度，彻底避免来源类型切换造成控件位移。
- 移除预览区渐变，滑条改为浅色轨道和橙色滑块，增加横屏/竖屏预览切换。
- 重做记录区显示结构，将状态、文件名、路径分开显示，避免中文状态被长路径挤成逐字换行。
- 修复文字水印在 ffmpeg 输出中出现问号的问题，处理时优先指定 macOS 可覆盖中英文的字体文件。
- 新增左侧视频栏批量拖拽导入，拖入时会过滤非视频文件并按路径去重。
- 修复预计时间逻辑：顶部只显示整批剩余时间，单个视频显示自身处理百分比。
- 预计时间优先读取 ffmpeg `speed`，早期采样不足时显示“估算中”，避免出现异常过长估算。
- 增加视频路径工具、文字水印字体、进度 speed 解析和批量导入相关测试。
